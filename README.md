# DropConvert

A native macOS menu bar app for quick file conversion.

Drag a file toward the menu bar icon — the panel springs open. Drop it in, pick your format, convert. That's it.

## Supported conversions

- HEIC → JPG / PNG
- MOV → MP4
- PDF → JPG / PNG

## How to run

1. Open `Package.swift` in Xcode
2. Select the `DropConvert` scheme and your Mac as the target
3. Run (`⌘R`)

The app will appear in your menu bar with no dock icon.

## Requirements

- macOS 14 (Sonoma) or later
- Xcode 15+
